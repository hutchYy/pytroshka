# pytroshka

## Simple uncompression tool to counter Matroshka CTF challenge written in python3

## Supportorted archives format :
- Zip
- Tar
- Bzip
- Gzip

## How to install ?

### Linux (easiest)

    git clone https://github.com/hutchYy/pytroshka
    pip3 install -r requirements.txt
    python3 pytroshka -h
